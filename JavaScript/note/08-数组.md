# （八）数组

## 1. 创建数组

### 1.1 new 创建对象

```js
let arr = new Array();
```

### 1.2 数组字面量创建数组

数组中可以存放任意数据类型。

```js
let arr = [];
```

## 2. 获取数组元素

索引访问。

```js
arr[index];
```

### 3. 遍历

```js
for (let i = 0; i < arr.length; i++) {
    // some code
}
```

### 4. 数组长度

```js
console.log(arr.length);
```
